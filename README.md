Игра представляет собой подобие Dinosaur Game от Google.

## Использование

Для запуска игры потребуется зависимость pygame:

```sh
$ pip install pygame
```

Далее склонируйте репозиторий и запустите main.py

Для Debian-like систем:

```sh
$ python3 main.py
```

Для Windows:

```sh
> python main.py
```
